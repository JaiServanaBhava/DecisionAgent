## 🧠 Overview

This project is a **next-generation AI agent system** designed with a **skill-based architecture**.  
It operates entirely within a GitHub repository, making it simple, transparent, and developer-friendly.

### ✨ Capabilities

- 🔍 Deep analysis of user situations  
- ⚠️ Identifying risks and constraints  
- 💡 Generating actionable advice  
- 🔮 Simulating possible outcomes  

---

## ⚙️ How It Works

The agent is structured into **independent skills**, each responsible for a specific type of reasoning.

Each skill:
- 📄 Has its own logic (`SKILL.md`)  
- 🧠 Uses structured reasoning steps  
- 🔗 Works together with other skills to produce intelligent outputs
