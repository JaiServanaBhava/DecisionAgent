<blockquote>

<div class="section">
<h2>🧠 Overview</h2>
<p>This project is a <b>next-generation AI agent system</b> designed with a <b>skill-based architecture</b>.
It operates entirely within a GitHub repository, making it simple, transparent, and developer-friendly.</p>

<p>The agent is capable of:</p>
<ul>
<li>🔍 Deep analysis of user situations</li>
<li>⚠️ Identifying risks and constraints</li>
<li>💡 Generating actionable advice</li>
<li>🔮 Simulating possible outcomes</li>
</ul>
</div>

<div class="section">
<h2>⚙️ How It Works</h2>
<p>The agent is structured into <b>independent skills</b>, each responsible for a specific type of reasoning.</p>

<ul>
<li>Has its own logic (SKILL.md)</li>
<li>Uses structured reasoning steps</li>
<li>Works together with other skills to produce intelligent outputs</li>
</ul>
</div>
