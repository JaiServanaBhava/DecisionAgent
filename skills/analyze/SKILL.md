# Analyze Situation

## Steps

1. Extract:
   - User goal
   - Available time
   - Financial condition
   - Skills and experience

2. Identify constraints:
   - Limitations
   - Missing resources

3. Detect hidden issues:
   - Unrealistic expectations
   - Skill gaps

4. Output a clear summary of the situation
